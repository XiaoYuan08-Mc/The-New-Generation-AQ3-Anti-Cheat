# 新一代反作弊AQ3

## 项目简介

新一代反作弊AQ3是一个面向Minecraft Paper服务器的先进反作弊插件。它采用了精确的运动模拟引擎，能够1:1复制玩家的所有可能动作，包括基础移动、游泳、击退、蜘蛛网效果、气泡柱以及乘坐船、猪、炽足兽等实体。

本项目已完全开源，代码托管在 [GitHub](https://github.com/XiaoYuan08-Mc/The-New-Generation-AQ3-Anti-Cheat)。

## 核心特性

### 精确运动模拟
- 实现完整的物理引擎，模拟所有玩家行为
- 覆盖从基本行走到复杂交互的所有动作
- 精确处理边缘案例以确保准确性

### 跨版本兼容性
- 支持多种客户端-服务器版本组合
- 自动处理版本间的技术差异
- 正确处理碰撞箱在版本间的差异

### 高性能架构
- 异步和多线程设计
- 世界复制系统支持并发处理
- 延迟补偿机制

### 全面检测类别
- 移动类作弊（速度、飞行、跳跃等）
- 战斗类作弊（杀戮光环、攻击距离等）
- 玩家类作弊（无摔落伤害、时间加速等）
- 物品类作弊（快速使用等）
- 世界类作弊（自动建造等）
- 基于机器学习的复杂作弊模式

### 硬件标识检测（HWID）
- 自动生成并跟踪玩家的硬件标识
- 防止被封禁玩家通过创建小号绕过封禁
- 支持关联账户检测和批量封禁

### AdvancedBan集成
- 与AdvancedBan插件无缝集成
- 支持永久封禁和临时封禁
- 支持踢出玩家功能
- 可配置的封禁和踢出原因
- 与内置封禁系统兼容的回退机制

## 系统架构

### 物理引擎 (PhysicsEngine)
模拟玩家在游戏中的物理行为，包括：
- 重力计算
- 跳跃机制
- 移动速度
- 环境影响（水、蜘蛛网、蜂蜜块、灵魂速度方块等）

### 世界管理系统 (WorldManager)
为每个玩家维护独立的世界副本：
- 实时同步方块变更
- 多线程安全访问
- 内存优化存储

### 检查系统 (CheckManager)
管理所有反作弊检查：
- 移动检查（速度、飞行、水上行走、相位、速度异常等）
- 战斗检查（杀戮光环、攻击距离、自动点击、暴击等）
- 物品检查（快速使用、背包移动等）
- 世界检查（脚手架、快速破坏等）
- 玩家检查（无摔落伤害、生命值恢复、聊天行为等）
- 机器学习模式分析检查

### 违规管理 (ViolationManager)
跟踪和管理玩家违规行为：
- 记录每次违规
- 统计违规次数
- 触发相应处罚

### 配置管理 (ConfigManager)
管理插件配置：
- 支持从YAML文件加载配置
- 可动态调整检查参数
- 支持启用/禁用特定检查

### 处罚管理 (PunishmentManager)
处理违规处罚：
- 根据违规次数执行不同处罚
- 支持踢出和封禁等处罚方式
- 支持硬件标识封禁以防止小号绕过

### 日志系统 (CheatLogger)
记录作弊行为：
- 记录详细作弊日志
- 支持系统事件日志
- 便于审计和分析

### 硬件标识管理 (HWIDManager)
防止通过创建小号绕过封禁：
- 自动生成并跟踪玩家硬件标识
- 支持封禁特定硬件标识
- 检测和管理关联账户

## 安装说明

1. 将生成的jar文件放入服务器的plugins目录
2. 重启服务器
3. 插件将自动生成配置文件
4. 根据需要调整配置文件

## 命令

- `/aq3` - 主命令
- `/aq3 reload` - 重新加载配置
- `/aq3 version` - 显示版本信息
- `/aq3 hwid` - 硬件标识管理命令

## 权限

- `aq3.admin` - 管理员权限

## 配置

配置文件将在首次启动时生成，您可以根据服务器需求进行调整。

### 基础配置

系统支持基础配置文件，可以设置：
- 检查启用/禁用
- 灵敏度设置
- 处罚阈值

### 管理员配置

系统支持管理员配置文件，可以设置：
- 管理员列表
- 免疫玩家列表
- 处罚设置
- 硬件标识设置

### AdvancedBan集成配置

系统支持与AdvancedBan插件的集成，可以设置：
- 启用/禁用AdvancedBan集成
- 临时封禁时长
- 自定义封禁和踢出原因

## 硬件标识功能

AQ3反作弊系统包含先进的硬件标识（HWID）功能，可以防止被封禁的玩家通过创建小号绕过封禁。

### 功能特性

1. **自动硬件标识生成** - 系统会自动为每个玩家生成唯一的硬件标识
2. **关联账户检测** - 可以检测使用相同硬件标识的多个账户
3. **封禁联动** - 封禁玩家时可选择同时封禁其硬件标识
4. **命令行管理** - 提供命令行工具管理硬件标识

### 命令

- `/aq3 hwid list` - 列出所有被封禁的硬件标识
- `/aq3 hwid ban <hwid>` - 封禁指定的硬件标识
- `/aq3 hwid unban <hwid>` - 解封指定的硬件标识
- `/aq3 hwid check <player>` - 检查玩家的硬件标识
- `/aq3 hwid info <hwid>` - 显示硬件标识的详细信息

## 开发

该项目采用Maven构建系统，包含以下模块：

1. `aq3-anticheat-common` - 核心反作弊逻辑
2. `aq3-anticheat-bukkit` - Bukkit平台适配
